Cyber Security Project

1) Phishing Attack

I have created the phishing attack login page — it’s the login page of the social media app Snapchat. When the user enters the password and email address and tries to sign in, the credentials will be captured in the AWS Console. After login, the user sees an acknowledgement page showing “502 Bad Gateway”.

2) VPN Server Setup

The VPN server setup involves creating two EC2 instances — one for the VPN server and one for the app server. The login page's IP address won't run on the browser of the machine used to create these instances. It must be run on the RDP setup window.

3) Load Balancer

Two instances are created named serverone and servertwo. A load balancer named serveronetwo balances the load between them. The two login pages are combined, and when we press CTRL + R, they switch.
